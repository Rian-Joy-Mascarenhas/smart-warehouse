About This System

The Smart Warehouse and Invoice Management System is a comprehensive solution designed for modern warehouses and businesses that need to manage inventory, sales, and invoicing efficiently.

Technology Stack:
Frontend: HTML, CSS, JavaScript (Vanilla)
Backend: Python, Flask
Database: MongoDB
Authentication: JWT (JSON Web Tokens)

This system combines ease of use with powerful features to help you manage your warehouse operations effectively.
